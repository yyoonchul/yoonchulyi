# 개인 포트폴리오 웹사이트

GitHub Pages로 호스팅되는 개인 포트폴리오 웹사이트입니다.

## 🚀 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화
- **모던 UI**: 깔끔하고 현대적인 디자인
- **부드러운 스크롤**: 네비게이션 링크 클릭 시 부드러운 스크롤
- **모바일 네비게이션**: 햄버거 메뉴로 모바일 친화적 네비게이션
- **애니메이션**: 스크롤 시 요소들이 부드럽게 나타나는 효과
- **SEO 최적화**: 메타 태그와 시맨틱 HTML 구조

## 📁 파일 구조

```
├── index.html          # 메인 HTML 파일
├── styles.css          # CSS 스타일시트
├── script.js           # JavaScript 기능
├── README.md           # 프로젝트 설명서
└── favicon.ico         # 파비콘 (선택사항)
```

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript (ES6+)**: 인터랙티브 기능
- **GitHub Pages**: 무료 호스팅

## 📝 사용법

1. **로컬에서 테스트**:
   ```bash
   # 프로젝트 폴더에서 간단한 HTTP 서버 실행
   python -m http.server 8000
   # 또는
   npx serve .
   ```

2. **GitHub Pages 배포**:
   - 이 저장소를 GitHub에 푸시
   - Settings > Pages에서 소스를 선택
   - `main` 브랜치의 루트 폴더를 선택

## 🎨 커스터마이징

### 개인 정보 수정
- `index.html`에서 이름, 소개, 프로젝트 정보 수정
- 연락처 링크 업데이트 (이메일, GitHub, LinkedIn 등)

### 스타일 수정
- `styles.css`에서 색상, 폰트, 레이아웃 수정
- CSS 변수를 사용하여 테마 색상 쉽게 변경 가능

### 기능 추가
- `script.js`에서 새로운 인터랙티브 기능 추가
- 폼 검증, 애니메이션 효과 등 확장 가능

## 📱 반응형 브레이크포인트

- **데스크톱**: 1200px 이상
- **태블릿**: 768px - 1199px
- **모바일**: 767px 이하
