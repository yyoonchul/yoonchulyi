## 📋 Quick Summary

### I think Anthropic and OpenAI have found product-market fit
**Source:** Simon Willison's Weblog · **Category:** AI · **Link:** [Original](https://simonwillison.net/2026/May/27/product-market-fit/)
- Simon Willison argues that Anthropic and OpenAI have found product-market fit through coding and general-purpose agent products such as Claude Code, Claude Cowork, and Codex.
- The core evidence is enterprise pricing: both vendors have moved enterprise agent usage toward listed API token pricing, turning heavy workplace use into much larger revenue than flat subscriptions.
- Willison treats April 2026 as a revenue inflection point, with higher-priced frontier models, enterprise API pricing, large hiring pushes, and budget shock stories all pointing in the same direction.

---

## 📝 Detailed Notes

### 1. I think Anthropic and OpenAI have found product-market fit

1. Willison frames coding agents as the first clear revenue fit for frontier AI labs.
   - Willison starts from rumors that Anthropic may be approaching a first profitable quarter and from reports of companies surprised by rising LLM bills.
   - The claim is not only that ChatGPT and Claude are popular, but that workplace agent usage has become economically serious.
   - Claude Code, Claude Cowork, and OpenAI Codex are presented as the products where usage, willingness to pay, and enterprise need now line up.
   - The article treats this as a different kind of fit from consumer ChatGPT's viral growth in February 2023, because the current pattern can produce much larger revenue per user.
2. Personal usage numbers show why flat subscriptions were unusually cheap for heavy agent users.
   - Willison pays $100 per month for Anthropic Max and $100 per month for OpenAI Pro.
   - A local `ccusage` estimate says the last 30 days would have cost $1,199.79 for Claude Code and $980.37 for Codex at API token prices.
   - That implies $2,180.16 of token usage for $200 of subscription spend.
   - Willison says this was not constant round-the-clock usage, which makes the implied enterprise exposure more important.
3. Enterprise pricing changes move large customers from seats toward API-like token bills.
   - Anthropic reportedly changed Enterprise from broad included usage to $20 per seat per month plus API pricing, with an Anthropic spokesperson placing the change in November 2025.
   - OpenAI's Codex rate card says pricing changed on April 2, 2026 to align with API token usage for Plus, Pro, Business, and new Enterprise plans.
   - OpenAI extended the change on April 23, 2026 to existing Enterprise plans, including Edu, Health, Gov, and ChatGPT for Teachers.
   - Willison concludes that by April 2026, enterprise use of Codex and Claude Code or Cowork effectively maps to listed API prices.
4. The business model becomes more plausible when agent users spend hundreds of dollars per month.
   - OpenAI said ChatGPT had more than 900 million weekly active users in February, but only 50 million paying consumer subscribers, or about 5.6%.
   - Willison argues that $10-$20 consumer subscriptions would need an enormous user base and long retention to cover infrastructure spending near $1 trillion.
   - Companies paying $200 or more per user per month can move the revenue equation faster.
   - The reason is that coding agents burn far more tokens while becoming daily tools for highly paid professionals and adjacent knowledge workers.
5. Hiring, budget pressure, and compute spending all support the product-market fit thesis.
   - Willison counts 703 OpenAI openings, with 229 or 32.6% categorized as enterprise sales and support related.
   - Anthropic has 390 openings, with 105 or 26.9% looking enterprise-oriented by Willison's classification.
   - Uber's AI budget overrun and Microsoft's Claude Code license reductions are read less as AI failure stories and more as evidence that customers value the tools enough to debate the bill.
   - A disclosed SpaceX cloud services agreement has Anthropic paying $1.25 billion per month through May 2029 for compute capacity, suggesting that inference demand has become extremely large.

## 📝 상세 정리

### 1. I think Anthropic and OpenAI have found product-market fit

1. Willison은 coding agent를 frontier AI lab의 첫 선명한 revenue fit으로 본다.
   - Willison은 Anthropic이 첫 profitable quarter에 가까워졌다는 rumor와, 기업들이 커지는 LLM bill에 놀라고 있다는 report에서 출발한다.
   - 핵심 주장은 ChatGPT와 Claude가 인기 있다는 수준이 아니라, workplace agent usage가 경제적으로 진지한 규모가 됐다는 점이다.
   - Claude Code, Claude Cowork, OpenAI Codex는 usage, willingness to pay, enterprise need가 맞물린 제품으로 제시된다.
   - 글은 이것을 2023년 2월 consumer ChatGPT의 viral growth와 다른 종류의 fit으로 본다. 지금의 패턴은 user당 훨씬 큰 revenue를 만들 수 있기 때문이다.
2. 개인 사용량 숫자는 heavy agent user에게 flat subscription이 얼마나 싼 구조였는지 보여준다.
   - Willison은 Anthropic Max에 월 $100, OpenAI Pro에 월 $100을 낸다.
   - 로컬 `ccusage` 추정치에 따르면 최근 30일 사용량은 API token 가격 기준 Claude Code $1,199.79, Codex $980.37에 해당한다.
   - 이는 $200 subscription으로 $2,180.16어치 token usage를 쓴 셈이다.
   - Willison은 이 사용량이 24시간 내내 agent를 돌린 결과도 아니었다고 말하며, 그래서 enterprise exposure가 더 중요해진다고 본다.
3. Enterprise pricing 변화는 대형 고객을 seat 중심 요금에서 API 같은 token bill로 옮긴다.
   - Anthropic은 Enterprise를 넓은 included usage에서 seat당 월 $20 plus API pricing으로 바꾼 것으로 알려졌고, Anthropic spokesperson은 그 변경 시점을 2025년 11월로 설명했다.
   - OpenAI의 Codex rate card는 2026년 4월 2일 Plus, Pro, Business, new Enterprise plan에 대해 API token usage와 맞추는 pricing 변경이 있었다고 적는다.
   - OpenAI는 2026년 4월 23일 기존 Enterprise plan에도 이 변경을 확장했고, Edu, Health, Gov, ChatGPT for Teachers까지 포함했다.
   - Willison은 2026년 4월 기준 Codex와 Claude Code 또는 Cowork의 enterprise usage가 사실상 공개 API price에 연결됐다고 결론낸다.
4. Agent user가 월 수백 달러를 쓰면 사업 모델은 더 그럴듯해진다.
   - OpenAI는 2월 ChatGPT weekly active user가 9억 명을 넘었다고 밝혔지만, paying consumer subscriber는 5천만 명, 약 5.6%였다.
   - Willison은 월 $10-$20 consumer subscription만으로는 약 $1 trillion infrastructure spending을 감당하려면 엄청난 user base와 긴 retention이 필요하다고 본다.
   - 반면 기업이 user당 월 $200 이상을 내면 revenue equation이 훨씬 빨리 움직인다.
   - 이유는 coding agent가 훨씬 많은 token을 태우면서도 고소득 professional과 인접 knowledge worker의 daily tool이 되고 있기 때문이다.
5. 채용, budget pressure, compute spending은 모두 product-market fit 논지를 뒷받침한다.
   - Willison은 OpenAI open job 703개 중 229개, 즉 32.6%를 enterprise sales와 support 관련으로 분류했다.
   - Anthropic은 open job 390개 중 105개, 즉 26.9%가 Willison의 분류상 enterprise-oriented로 보인다.
   - Uber의 AI budget overrun과 Microsoft의 Claude Code license reduction은 AI failure story라기보다, 고객이 bill을 두고 논쟁할 만큼 도구를 가치 있게 본다는 증거로 읽힌다.
   - 공개된 SpaceX cloud services agreement에는 Anthropic이 2029년 5월까지 compute capacity 대가로 월 $1.25 billion을 내는 내용이 있어, inference demand가 매우 커졌다는 신호로 해석된다.
