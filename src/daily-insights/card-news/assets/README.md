Replace this file's sibling `background.*` image to swap the card-news background.

Supported filenames:
- `background.png`
- `background.jpg`
- `background.jpeg`
- `background.webp`
- `background.svg`

How it works:
- The templates automatically look for the first matching `background.*` file in this folder.
- `cover`, `summary`, and `insight` all share the same background image.
- If no background image exists here, the templates fall back to the built-in dark editorial background.
